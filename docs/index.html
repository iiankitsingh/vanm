Integration API Documentation

Integrate our parsed Navi Mumbai airspace JSON dataset directly into your Electronic Flight Bag (EFB), flight simulators, or local databases.

GET /notams.json Returns a compiled list of active and parsed NOTAMs for the Mumbai region.

Example JavaScript Request

fetch('notams.json')
  .then(res => res.json())
  .then(notams => {
    console.log(`Found ${notams.length} active flight notices.`);
    notams.forEach(item => {
      console.log(`[${item.priority}] ${item.id}: ${item.plain_text}`);
    });
  });
