GraphQL API Endpoint

/graphql

AeroParse exposes a standard Strawberry GraphQL interface at /graphql. You can copy these queries and run them in any GraphiQL playground.

1. Query Active NOTAMs by Coordinates & Radius (NM)

query {
  notams(lat: 40.6397, lon: -73.7789, radiusNM: 10.0) {
    id
    location
    priority
    plainText
    validFrom
    validTo
    geometry {
      type
      coordinatesJson
    }
  }
}

2. Real-time Subscription for New NOTAM Ingests

subscription {
  notamAdded {
    id
    location
    plainText
    priority
  }
}
